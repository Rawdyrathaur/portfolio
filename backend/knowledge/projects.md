# Projects of Manish

## Project: Tab Story

### Project Overview
Tab Story is a Chrome browser extension developed by Manish. Tab Story intelligently organizes browser tabs using automatic intent detection and AI-powered summaries. Tab Story helps users manage multiple tabs efficiently by grouping them based on browsing purpose.

### What Tab Story Does
Tab Story automatically groups browser tabs based on user intent such as job search, learning, or shopping. Tab Story uses Chrome's built-in AI capabilities to analyze open tabs and generate meaningful groups. Tab Story also provides AI-generated summaries of webpages to help users quickly understand content.

### Key Features of Tab Story
Tab Story provides smart tab grouping based on browsing intent and topic. Tab Story includes intent detection to identify what the user is working on. Tab Story generates AI-powered summaries using Chrome’s local AI processing. Tab Story includes a timeline view to track browsing sessions by time and intent. Tab Story allows instant search across tab titles, notes, and intent labels. Tab Story automatically backs up tab groups and ensures user privacy by processing AI tasks locally on the device.

### Technology Stack
Manish built Tab Story using JavaScript and CSS for the Chrome extension frontend and interface logic. Manish used Python for AI-related logic and processing. Tab Story integrates with Chrome’s built-in AI capabilities for summarization and intent detection.

### Role of Manish
Manish developed Tab Story as a solo project. Manish designed the architecture, implemented the AI-powered tab organization logic, and built the user interface. Manish handled development, testing, publishing, and marketing of the extension independently.

### Project Status
Tab Story is completed and publicly available on the Chrome Web Store. Manish is actively focusing on improving the product and increasing user adoption through marketing and feature enhancements.
